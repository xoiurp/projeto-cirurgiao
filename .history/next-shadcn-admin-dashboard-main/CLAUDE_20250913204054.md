## Project Overview

Studio Admin is a Next.js 15 admin dashboard template built with TypeScript, Tailwind CSS v4, and shadcn/ui components. It features a colocation file system architecture where pages, components, and logic are grouped by feature.

## Development Commands

```bash
# Development server with Turbopack
npm run dev

# Production build
npm run build

# Start production server
npm start

# Linting
npm run lint

# Code formatting
npm run format
npm run format:check

# Generate theme presets
npm run generate:presets
```

## Architecture

### File Structure
- **Colocation Pattern**: Features are grouped in their own directories with all related components, styles, and logic
- **Route Groups**: Uses Next.js app router with route groups `(main)` and `(external)`
- **Component Organization**: 
  - `src/components/ui/` - Reusable shadcn/ui components
  - `src/components/data-table/` - Data table components with TanStack Table
  - Route-specific components in `_components/` folders

### Key Directories
- `src/app/(main)/dashboard/` - Main dashboard layouts and pages
- `src/app/(main)/auth/` - Authentication screens (v1 and v2 variants)
- `src/stores/` - Zustand state management
- `src/config/` - Application configuration
- `src/types/` - TypeScript type definitions
- `src/server/` - Server actions

### State Management
- **Zustand**: Primary state management for preferences and UI state
- **Server Actions**: Used for theme/layout preferences persistence
- **React Hook Form + Zod**: Form handling and validation

### Theme System
- Multiple theme presets (default, tangerine, brutalist, soft-pop)
- Theme mode (light/dark) stored in cookies
- Custom CSS properties for theming
- Theme preferences managed via server actions

### UI Components
- **shadcn/ui**: Core component library with "new-york" style
- **Lucide React**: Icon library
- **TanStack Table**: Advanced data tables with drag/drop
- **Recharts**: Data visualization
- **Sonner**: Toast notifications

### Path Aliases
- `@/*` maps to `src/*`
- `@/components` for UI components
- `@/lib/utils` for utilities
- `@/hooks` for custom hooks

## Tech Stack Details
- **Next.js 15**: App router, React Server Components
- **TypeScript**: Strict mode enabled
- **Tailwind CSS v4**: Utility-first CSS with custom properties
- **ESLint**: Code linting with Next.js config
- **Prettier**: Code formatting with Tailwind plugin
- **Husky**: Git hooks for pre-commit linting
